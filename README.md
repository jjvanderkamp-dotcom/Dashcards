# DashCards

Minimalist notification icons designed for Home Assistant dashboards, wall panels and Google Hub casting.

DashCards focuses on simple, clean and easy-to-recognize icons that work well on dark dashboards and smart displays.

---

## Features

- Colored backgronds
- Minimal monochrome design
- White icons optimized for dark themes
- Consistent visual style
- 1020x600 PNG format 
- Designed for Home Assistant
- Great for Google Nest Hub casting (perfect size)

## Design Style

DashCards are intentionally designed to be:

- Simple
- Minimal
- Recognizable at a glance
- Consistent across all icons

Every icon uses:
- Transparent background
- Single-color design
- Large readable shapes
- Minimal detail
